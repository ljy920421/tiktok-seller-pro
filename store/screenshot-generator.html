How to capture screenshots:

  1. Open Chrome DevTools (F12)
  2. Press Ctrl+Shift+P, type "screenshot"
  3. Choose "Capture node screenshot"
  4. Select the .screenshot div in Elements panel
  5. Or: right-click each screenshot frame > Inspect > right-click element > "Capture node screenshot"
